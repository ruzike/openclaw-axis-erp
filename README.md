# OpenClaw AXIS ERP Framework

Это готовая сборка из 12 ИИ-агентов (Директор, DevOps, Продажи, HR, Контроль Качества и др.), работающих как единая система управления бизнесом (ERP) на базе фреймворка [OpenClaw](https://openclaw.ai).

## 🏢 Что внутри?
- **12 специализированных агентов** с прописанными ролями (`SOUL.md`), зонами ответственности и инструкциями по делегированию задач.
- **Интеграция с Trello** (двусторонний обмен): создание задач, отслеживание дедлайнов, ежедневные отчеты.
- **Система дашбордов** (автогенерируемые HTML страницы с KPI и ритуалами команды).
- **Скрипты автоматизации:** мониторинг системы (watchdog), бэкапы, векторная база знаний (ChromaDB).
- **Многопользовательский Telegram-интерфейс** (каждый сотрудник общается с агентами через Telegram-ботов).

## 🚀 Быстрый старт
```bash
git clone https://github.com/ruzike/openclaw-axis-erp.git
cd openclaw-axis-erp
./install.sh
```

## 🧠 Архитектура
Каждый агент имеет свою директорию (например `~/.openclaw/agents/sales/`):
- `SOUL.md` — личность и правила поведения
- `AGENTS.md` — матрица ответственности и правила передачи задач
- `MEMORY.md` — долгосрочная память агента
- `memory/core/` — знания, уроки и паттерны ошибок

Агенты общаются между собой, передавая задачи (Handoff), и отчитываются в общий чат.
