## 🔧 СИСТЕМНЫЕ CRON-ЗАДАЧИ (crontab -l)

### 1. AXIS System Monitor
| Поле | Значение |
|------|----------|
| **Название** | AXIS System Monitor |
| **Агент-владелец** | DevOps |
| **Расписание** | `*/15 * * * *` — каждые 15 минут, 24/7 |
| **Скрипт** | `axis-system/monitor.py` |
| **Лог** | `/tmp/monitor.log` |
| **Описание** | Мониторинг здоровья системы: проверка логов cron, ошибок, статуса интеграций Trello, активности сотрудников. Эскалирует критические ошибки Руслану. |
| **Статус** | 🟢 АКТИВЕН |
