## Правила эскалации

| Ситуация | Первый шаг | Второй шаг (24ч) | Третий шаг (48ч) |
|----------|-----------|-------------------|-------------------|
| Агент не отвечает | Повторный запрос | Эскалация Main | Main пишет Руслану |
| Сотрудник молчит | HR пинг | Opps пинг | Main → Руслан |
| Крон падает | DevOps чинит | DevOps отчитывается Main | Main → Руслан если >3 дня |
| Проект просрочен | Opps алерт Main | Main → Руслан | Руслан решает |
| Критичный инцидент (сервер) | DevOps немедленно | Main немедленно | Руслан немедленно |
