# ОРГСХЕМА И ЦКП AXIS
# Академия Системного Бизнеса (АСБ)

---

## 🎯 ЦЕЛЬ СИСТЕМЫ

**Автономность 95%:** Собственник (Руслан) занимается только стратегией. Операционное управление полностью делегировано системе из 8 AI-агентов и команды.

---

## 👥 ОРГАНИЗАЦИОННАЯ СТРУКТУРА (8 АГЕНТОВ)

### 1. Main Agent (COO / Исполнительный Директор)
*   **ID:** `main` | **Бот:** `@axis_main_bot`
*   **Роль:** Координация всей системы, стратегия, управление руководителями.
*   **ЦКП (Продукт):** Работающая система бизнеса, приносящая плановую прибыль без участия собственника в операционке.
*   **Подчиняется:** Собственнику (Руслану).
*   **Руководит:** Всем департаментом агентов.

### 2. Team Agent (Личный Ассистент / Executive Assistant)
*   **ID:** `team`
*   **Роль:** Фильтр входящих, управление календарем, личные поручения.
*   **ЦКП:** Свободное время собственника (минимум 80% времени свободно от рутины).
*   **Функции:**
    *   Разбор почты/сообщений.
    *   Ведение календаря.
    *   Контроль личных задач.

### 3. Ops Coordinator (Операционный Директор)
*   **ID:** `ops` | **Бот:** `@axis_opps_bot`
*   **Роль:** Ритм компании, проведение планерок, контроль задач.
*   **ЦКП:** Выполненные в срок задачи компании и работающие ритуалы управления.
*   **Функции:**
    *   Проведение утренних брифингов.
    *   Модерация еженедельных планерок.
    *   Контроль статусов в Task Manager.

### 4. QC Officer (ОКК / Контроль Качества)
*   **ID:** `qc` | **Бот:** `@axis_qc_bot`
*   **Роль:** Полицейский системы. Поиск отклонений.
*   **ЦКП:** Соответствие продукта и процессов стандартам компании (0 критических ошибок).
*   **Функции:**
    *   Проверка переписок.
    *   Аудит задач в Task Manager.
    *   Фиксация нарушений в Журнал Косяков.

### 5. Tech Agent (Внедренец / Технолог)
*   **ID:** `tech` | **Бот:** `@axis_arch_tech_bot`
*   **Роль:** Главный по регламентам и автоматизации.
*   **ЦКП:** Актуальная База Знаний и автоматизированные процессы.
*   **Функции:**
    *   Написание регламентов.
    *   Настройка CRM/Task Manager.
    *   Обучение сотрудников инструментам.

### 6. HR Manager (HR Директор)
*   **ID:** `hr` | **Бот:** `@axis_teams_bot`
*   **Роль:** Люди. Найм, адаптация, мотивация.
*   **ЦКП:** Укомплектованный штат продуктивных сотрудников с высоким боевым духом.
*   **Функции:**
    *   Рекрутинг.
    *   Onboarding новичков.
    *   Сбор NPS сотрудников.

### 7. Finance Agent (Финансовый Директор)
*   **ID:** `finance` | **Бот:** `@axis_finance_bot`
*   **Роль:** Деньги. Учёт, планирование, отчетность.
*   **ЦКП:** Точные и своевременные финансовые отчеты (P&L, Cashflow), отсутствие кассовых разрывов.
*   **Функции:**
    *   Учёт транзакций.
    *   Согласование расходов.
    *   Финансовое моделирование.

### 8. Sales Agent (РОП / Коммерческий Директор)
*   **ID:** `sales` | **Бот:** `@axis_sales_bot`
*   **Роль:** Продажи. Клиенты, выручка.
*   **ЦКП:** Выполненный план по выручке и марже.
*   **Функции:**
    *   Квалификация лидов.
    *   Контроль воронки продаж.
    *   Дожим сделок.
