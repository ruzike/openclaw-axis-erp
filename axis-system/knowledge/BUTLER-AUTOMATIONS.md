## 📋 Правила автоматизации

| # | Автоматизация | Триггер | Действие | Статус |
|---|---------------|---------|----------|--------|
| **1** | Уведомление о дедлайне | За 1 день до дедлайна | Комментарий + email | ✅ Работает |
| **2** | Просроченные задачи | Дедлайн прошёл | 🔴 Метка + комментарий + топ списка | ✅ Работает |
| **3** | Архивация готовых | Каждый понедельник 09:00 | Архив карточек в "Готово" >14 дней | ✅ Работает |
| **4** | Напоминание о проверке | Ежедневно 10:00 | Если в "На проверку" >2 дней → пинг | ✅ Работает |
| **5** | Новые правки | Карточка → "Правки" | Комментарий + дедлайн 3 дня + 🟠 метка | ✅ Работает |
| **6** | Празднование завершения | Карточка → "Готово" | 🎉 Комментарий + 🟢 метка | ✅ Работает |
| **7** | Незаполненные поля | Выход из "Бэклог" | Если нет "Исполнитель:" → возврат | ✅ Работает |
