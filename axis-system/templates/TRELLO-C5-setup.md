# Промпт для OpenClaw.ai бота: Настройка Trello-доски для процесса C5

## Контекст
Ты настраиваешь доску Trello для дизайн-бюро архитектурных решений OpenClaw.ai.
Доска управляет процессом C5 «Авторский надзор» — контроль реализации проекта, выезды на объекты, фотофиксация, составление отчётов.

Методология: Академия Системного Бизнеса (АСБ), Модуль 3 «Регламенты и инструкции».

Принципы АСБ для Trello:
- Карточка = объект проекта (не разовая задача)
- Чеклист выезда = критерий завершения выезда
- Колонки = этапы работы с объектом
- Перемещение карточки = формальная передача ЦКП между этапами

Участники: Главный дизайнер (владелец процесса), Заказчик, Подрядчики, Архитектор.
Типы объектов: Жилые комплексы, Квартиры, Офисы, Бизнес-центры, Благоустройство.

**Название доски:** `C5 — Авторский надзор`

**Описание доски:**
```

Владелец процесса: Главный дизайнер.

Правила:
```

## Задача 4: Создать карточки-шаблоны

### Шаблон 1: Активный объект (общий)

**Название карточки:** `[ШАБЛОН] Активный объект — {Название ЖК / Объект}`

**Метка:** Активный (зелёная)

**Описание карточки:**
```
## Информация об объекте
- **Тип объекта:** {ЖК / Квартира / Офис / Территория}
- **Название:** {Название ЖК / Адрес / Название}
- **Статус реализации:** {Начало / В процессе / Критический}
- **Ответственный у заказчика:** {Имя}
- **Ответственный у подрядчика:** {Имя}
- **Последний выезд:** {дата}

## Процесс (по регламенту C5)
1. Создать карточку для объекта (если нет)
2. Запланировать выезды по графику
3. Перед каждым выездом:
   - Изучить рабочую документацию
   - Изучить утверждённую концепцию
   - Составить план осмотра
4. Провести выезд на объект
5. Сделать фотофиксацию
6. Сравнить фактическое состояние с концепцией
7. Зафиксировать все отклонения
8. Составить отчёт о выезде
9. Отправить отчёт заказчику
10. Получить подтверждение согласования
11. Обновить план исправлений

## Правила
⚠ Выезды проводить по графику (минимум 2 выезда в месяц)
⚠ Перед каждым выездом изучать документацию и концепцию
⚠ Фотофиксация минимум 5-10 фото на выезд
⚠ Все отклонения фиксировать в таблице (см. регламент C5)
```

**Чеклист 1 — «Подготовка к выезду»:**
```
☐ Рабочая документация изучена
☐ Утверждённая концепция изучена
☐ План осмотра составлен
☐ Элементы для проверки определены
☐ Фотоаппарат/смартфон подготовлен
☐ Ответственный у заказчика уведомлён
```

**Чеклист 2 — «Выезд на объект»:**
```
☐ Прибыть вовремя (за 5-10 минут до времени)
☐ Осмотреть объект согласно плану
☐ Сравнить фактическое состояние с концепцией
☐ Сделать фотофиксацию (минимум 5-10 фото)
☐ Провести диалог с подрядчиком (если нужно)
☐ Записать заметки в блокнот
```

**Чеклист 3 — «Анализ и фиксация отклонений»:**
```
☐ Все отклонения зафиксированы в таблице
☐ Определены приоритеты (высокий/средний/низкий)
☐ Высокие отклонения отмечены красным
☐ Средние отклонения отмечены жёлтым
☐ Низкие отклонения отмечены зелёным
```

**Чеклист 4 — «Отчёт о выезде»:**
```
☐ Отчёт о выезде составлен (текст + фото)
☐ План исправлений составлен
☐ План исправлений согласован с заказчиком
☐ Подтверждение согласования получено
☐ Карточка обновлена
```

После создания — **пометить карточку как шаблон** (Card Template).

## Задача 5: Настроить автоматизации (Butler)

Если доступен Trello Butler, настрой следующие правила:

### Автоматизация 1: Напоминание о выезде на сегодня
```
Триггер: Ежедневно 09:00
Действие: добавить комментарий "📅 Выезды на сегодня: {список выездов}"
```

### Автоматизация 2: Просроченный выезд
```
Триггер: Карточка в «Выезды (запланировано)» и дата наступила, но карточка не перемещена в «Отчёты»
Действие: добавить красную метку "ПРОСРОЧЕНО" и уведомление
```

### Автоматизация 3: Напоминание о неактивном объекте
```
Триггер: Карточка в «Объекты» без выездов > 14 дней
Действие: добавить комментарий "⚠ Нет выездов более 2 недель. Нужно запланировать выезд."
```

### Автоматизация 4: Уведомление при переносе в «Архив»
```
Триггер: Карточка перемещена в «Архив»
Действие: добавить комментарий "✅ Объект сдан: {дата}. Замерить метрики: % отклонений с высокой приоритетностью, % планов исправлений, количество выездов."
```
