# РЕГЛАМЕНТ C3 — Рабочая документация

**Версия:** 2.0 | **Владелец:** Главный дизайнер

---

## 1. ЦЕЛЬ ПРОЦЕССА

Разработать полный комплект рабочей документации, достаточный для реализации проекта на объекте без дополнительных пояснений.

---

## 2. ЦКП — Ценный Конечный Продукт

📌 ЦКП зависит от статуса карточки в Trello:

- Статус «В работе» → Черновой комплект чертежей, готовый к внутренней проверке
- Статус «На проверке» → Комплект прошёл ОКК без критических замечаний
- Статус «Готово» → PDF-альбом рабочей документации, ОКК 100%, передан заказчику

---

## 3. ВХОДЫ / ВЫХОДЫ

### Входы
- Утверждённая концепция из C1
- Утверждённые рендеры из C2 (если применимо)
- Обмерные планы / планы БТИ
- Спецификация материалов

### Выходы
- PDF-альбом рабочей документации (чертежи, спецификации, узлы)
- Чек-лист ОКК — заполнен на 100%
- Исходные файлы (DWG/ArchiCAD)

---

## 4. СХЕМА ПРОЦЕССА

```
КОНЦЕПЦИЯ/РЕНДЕРЫ → РАЗРАБОТКА ЧЕРТЕЖЕЙ → ВНУТРЕННЯЯ ПРОВЕРКА (ОКК) → КОРРЕКТИРОВКА → ФОРМИРОВАНИЕ PDF → ПЕРЕДАЧА ЗАКАЗЧИКУ

↳ ОКК пройден → PDF-альбом → Заказчик → C5 (Авторский надзор)
↳ ОКК не пройден → Возврат на доработку → Повторная проверка
```

## 6. АЛГОРИТМ

### Шаг 1: Подготовка (1 день)
- Определить состав альбома по типу объекта (МОП/Фасад/Благоустройство/Квартира)
- Подготовить шаблон альбома
- Распределить листы между исполнителями

### Шаг 2: Разработка чертежей (5–10 дней)
- Планы: демонтажа, монтажа, отделки полов, потолков, раскладки плитки
- Развёртки стен
- Узлы и детали
- Спецификации материалов

### Шаг 3: ОКК — Отдел Контроля Качества (1–2 дня)
Главный дизайнер проверяет по чек-листу ОКК:
- Полнота комплекта (все листы на месте)
- Соответствие утверждённой концепции
- Размеры и привязки (нет ошибок)
- Оформление по стандарту AXIS
- Спецификации заполнены

### Шаг 4: Корректировка (при необходимости)
- Устранение замечаний ОКК
- Повторная проверка

### Шаг 5: Формирование PDF и передача
- Экспорт в PDF
- Загрузка на Google Drive в папку проекта
- Отправка заказчику
- Перевод карточки в статус «Готово»

---

## 7. МЕТРИКИ

- Срок рабочей документации — ≤ 7 дней (МОП), ≤ 14 дней (Квартира)
- ОКК с первого раза — целевой ≥ 80%
- Критические ошибки в чертежах — 0
- Полнота альбома — 100% по чек-листу

---

## 8. ЭСКАЛАЦИЯ

- **Уровень 1:** Главный дизайнер → исполнитель не укладывается в сроки
- **Уровень 2:** Главный дизайнер → критическая ошибка обнаружена после передачи заказчику

**Правило:** Документация НЕ передаётся заказчику без прохождения ОКК.
