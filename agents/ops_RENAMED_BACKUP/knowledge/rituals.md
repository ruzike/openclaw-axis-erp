## 📋 ЕЖЕДНЕВНЫЕ РИТУАЛЫ (Пн-Пт)

### 09:00 — Утренний брифинг

**Задача:** Собрать план дня

**Шаги:**
1. Запустить `python3 /home/axis/openclaw/trello-briefing.py`
2. Проанализировать вывод
3. Публиковать план дня в общем чате

**Формат:**
```
📋 План дня (Понедельник, 09:00):

**Что сделано вчера:**
- [заполнить]

**Что планирую сегодня:**
1. [задача 1] — [ответственный]
2. [задача 2] — [ответственный]
3. [задача 3] — [ответственный]

**Блокеры:**
- [если есть]
```

**Контроль:**
- Проверить Trello (нет ли новых задач)
- Проверить Google Forms (заполнили ли?)

---

### 14:00 — Проверка зависших задач

**Задача:** Найти зависшие задачи

**Шаги:**
1. Запустить `python3 /home/axis/openclaw/team-manager.py triggers`
2. Проанализировать вывод
3. Если есть проблемы — написать в общий чат

**Формат:**
```
⚠️ Зависшие задачи:
1. [задача] — [ответственный] — [просрочен на X дней]
2. [задача] — [ответственный] — [нет активности]

Рекомендация: связаться с [ответственным]
```

**Контроль:**
- Если есть молчунов — сделать пинг
- Если есть просроченные задачи — напомнить

### 09:00 — Планёрка

**Задача:** Провести планёрку

**Длительность:** 60-90 минут

**Последовательность:**
1. Открыть Trello
2. Запросить фактические результаты
3. Анализировать "План/Факт"
4. Выявлять блокеры
5. Планировать на неделю
6. Устанавливать дедлайны
7. Фиксировать задачи в Trello

**Формат протокола:**
```
📋 Протокол планёрки (Понедельник, 09:00)

**Что из плана недели выполнено:**
- [задача] — ✅
- [задача] — ✅

**Блокеры:**
- [описание]

**Планы на неделю:**
1. [задача] — [ответственный] — [срок]
2. [задача] — [ответственный] — [срок]

**Ссылка на Trello:** [ссылка]
```

**Контроль:**
- Все регламенты соблюдаются
- Нет незакрытых задач с прошлого периода
- Каждая задача имеет ответственного и срок

---

### 10:30 — Фиксация новых задач в Trello

**Задача:** Обновить Trello с новыми задачами

**Шаги:**
1. Открыть Trello
2. Создать новые карточки
3. Назначить ответственных
4. Поставить сроки

**Формат карточки:**
```
**Задача:**
Краткое описание

**Ответственный:**
[имя]

**Срок:**
[дата]

**Описание:**
[подробное описание]

**Статус:**
- [ ] Сделал
- [ ] В процессе
```

## 📋 ЕЖЕМЕСЯЧНЫЕ РИТУАЛЫ

### Ежемесячная ретроспектива (последний день месяца)

**Длительность:** 2-3 часа

**Формат:**
```
📊 Ретроспектива (месяц X)

**Что из плана выполнено:**
- X задач закрыто
- Y задач открыто

**Плюсы недели:**
1. [пункт 1]
2. [пункт 2]

**Минусы недели:**
1. [пункт 1]
2. [пункт 2]

**Дельтаплюсы (чтобы минусы не повторились):**
1. [действие 1]
2. [действие 2]
```

**Контроль:**
- 2 плюса
- 2 дельтаплюса
- Все дельтаплюсы → карточки в Trello
