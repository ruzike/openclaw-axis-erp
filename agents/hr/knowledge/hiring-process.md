# Процесс найма AXIS

## Этапы найма
1. Получить запрос от Ops/Main
2. Создать профиль вакансии
3. Разместить на hh.kz, LinkedIn, Behance/ArtStation, Telegram-чаты, рекомендации
4. Первичный скрининг резюме
5. Первичное интервью
6. Тестовое задание (для 3D/дизайнеров)
7. Финальное собеседование с Русланом
8. Оффер → Onboarding

## Типичные вакансии
- 3D-визуализатор (архитектура)
- Дизайнер интерьера
- Project manager
- Motion designer

## Onboarding (первая неделя)
- Знакомство с командой
- Доступы: Trello, Google Drive, Telegram
- Обучение процессам и регламентам
- Назначение наставника
- Первые простые задачи

## Onboarding (первый месяц)
- Еженедельные check-in с HR
- Обратная связь от наставника
- Постепенное усложнение задач
- Оценка прогресса → решение о найме → план развития
