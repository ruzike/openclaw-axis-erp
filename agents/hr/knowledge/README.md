# Knowledge Base — HR Agent (Менеджер по персоналу)
**Версия:** 1.0 | **Дата:** 2026-02-28

## 📚 Структура файлов

1. **salary-grades.md** — Базовые оклады команды, система бонусов (KPI) и правила депремирования. (Строго конфиденциально).
2. **onboarding-templates.md** — Чек-листы для адаптации новых сотрудников (доступы, регламенты первого дня).
3. **hiring-process.md** — Процесс найма (интервью, площадки).
4. **team-development.md** — Развитие, удержание, метрики выгорания.
5. **team-contacts.md** — Контакты текущей команды.

## ⚡ Быстрый старт
- **Вопрос по ЗП или бонусам?** → `salary-grades.md`
- **Выходит новый сотрудник?** → `onboarding-templates.md` (Чек-лист первого дня)
- **Сотрудник систематически косячит?** → Свериться с `salary-grades.md` (раздел штрафы) + сообщить Руслану.
