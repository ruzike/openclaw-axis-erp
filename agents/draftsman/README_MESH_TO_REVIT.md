# Конвертер: 3ds Max → Revit

Автоматическое распознавание стен из 3D-mesh (полигональных сеток).

---

## 🚀 Быстрый старт

```bash
python3 scripts/mesh_to_revit_full.py
```

**Результат:** `temp/revit_commands.jsonl` (61 команда CreateWall)

---

## 📋 Что делает

1. Загружает 3D-модель из Speckle (проект `30eb4c8084`)
2. Анализирует 9 mesh-объектов
3. Распознаёт **249 стеновых граней** (вертикальные поверхности)
4. Группирует грани → **70 логических стен**
5. Фильтрует по высоте (> 2м) → **61 валидная стена**
6. Экспортирует JSONL команды для Revit Bridge

## ⚙️ Параметры

Редактировать в `mesh_to_revit_full.py`:

```python
MIN_WALL_HEIGHT = 2000    # Минимальная высота стены (мм)
WALL_THICKNESS = 200      # Толщина стены (мм)
LEVEL_NAME = "1 этаж"     # Имя уровня в Revit
```

---

## 📤 Импорт в Revit

1. **Скопировать файл:**
   ```cmd
   copy temp\revit_commands.jsonl C:\bridge\
   ```

2. **Запустить в Revit:**
   - pyRevit → Bridge → Load Commands
   - Или: напрямую через Python скрипт (если bridge настроен)

3. **Результат:**
   - 61 стена построена автоматически
   - Толщина: 200 мм
   - Уровень: "1 этаж"

## 🔧 Алгоритм

### 1. Распознавание стен (по грани)
```python
# Вертикальная грань: |нормаль.z| < 0.5
# Минимальная высота: 0.5м
# Минимальная ширина: 0.3м
```

### 2. Группировка граней → логическая стена
```python
# Похожие нормали: cos(угол) > 0.95
# Расстояние между центрами: < 3м
```

### 3. Определение начала/конца стены
```python
# Если нормаль ∥ X → стена ∥ Y
# Если нормаль ∥ Y → стена ∥ X
# Иначе: диагональная (начало = min, конец = max)
```

## 🛠 Отладка

**Посмотреть все команды:**
```bash
cat temp/revit_commands.jsonl | jq
```

**Статистика высот:**
```bash
cat temp/revit_commands.jsonl | jq '.height' | sort -n | uniq -c
```

**Топ-10 длинных стен:**
```bash
cat temp/revit_commands.jsonl | jq -r 'select(.endX != .startX or .endY != .startY) | "\(sqrt(pow(.endX - .startX, 2) + pow(.endY - .startY, 2))/1000)м × \(.height/1000)м"' | sort -rn | head -10
```

## ⚠️ Ограничения

| Поддерживается | Не поддерживается |
|----------------|-------------------|
| ✅ Прямоугольные стены | ❌ Криволинейные стены |
| ✅ Вертикальные грани | ❌ Двери и окна (проёмы) |
| ✅ Группировка триангуляции | ❌ Разные типы стен |
| ✅ Фильтр по высоте | ❌ Многоуровневые здания |
| ✅ Автоопределение направления | ❌ Навесные фасады |
