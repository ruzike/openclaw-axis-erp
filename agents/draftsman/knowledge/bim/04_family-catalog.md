## 2. Витражи и остекление

Окна в этой студии делаются как **витражи**, не стандартные окна Revit.

| Тип | Применение | Особенности |
|---|---|---|
| Стандартный витраж | Балконное остекление, большие проёмы | Схемы разрезки, импосты настраиваются |
| Простое окно | Окна с открыванием | Системная панель заменяется на нужный тип |
| Дверь витражная | Балконная дверь в витраже | Открепить панель → заменить на "Дверь витраж" |
| Раздвижная дверь | Раздвижная балконная дверь | Изменить импосты под ширину двери |
| Стеновая панель | Заглушка/утеплитель в витраже | Создаётся дубликат системной панели |

**Алгоритм замены панели в витраже:**
1. Tab → выделить системную панель (стекло)
2. "Открепить прикрепление"
3. Изменить тип → выбрать нужный (окно / дверь / стеновая панель)

## 3. Двери

| Тип | Параметры | Применение |
|---|---|---|
| Дверь однопольная (существующая) | Ширина, Высота, Толщина полотна | Только для обмера — менять размеры по факту |
| Дверь витражная | Подстраивается под панель витража | Балконная в витраже |

**Важно:** На плане монтажа двери скрываются фильтром `Двери для монтажа` — строители видят только проём.

Стандарт указания проёма на обмерном плане: `ширина проёма × высота проёма × высота от пола`

## 5. Группы элементов (Model Groups)

Готовые группы аннотаций в шаблоне, которые **дублируются на все планы**:

| Группа | Содержимое | Планы |
|---|---|---|
| Коллекторный узел | Штриховка + подписи шахты, труб, коллектора | Монтаж и все последующие |
| Шахта вентиляции | Условное обозначение шахты | Обмер, Демонтаж, Монтаж и далее |

**Как работать с группой:**
- На обмерном плане → обозначить шахту/коллектор
- Скопировать (`Ctrl+C`) → перейти на монтаж → "Вставить по текущему виду"
- Войти в группу → отредактировать под текущий план → выйти

## 8. Аннотации и легенды

| Тип | Применение |
|---|---|
| Высотная отметка (план) | Обмерный план, узлы |
| Высотная отметка (фасад) | Разрезы, развёртки |
| Марка категории | Автоматическая маркировка дверей, помещений |
| Маркировать все | Разом промаркировать все помещения на листе |
| Облако пометок | Замечания, ревизии |
| Текст с выноской | Подписи высот проёмов, вентиляции, коммуникаций |

**Стандарт выноски для проёма:**
```
Высота подоконника: 900
Высота проёма: 1900
```
Текст с выноской, галочка "Вверх" для верхней линии.
