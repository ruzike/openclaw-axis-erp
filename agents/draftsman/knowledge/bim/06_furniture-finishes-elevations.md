# Мебель, Отделка, Развёртки — Стандарт построения

> Источник: Модули 05-01, 05-03, 05-04 курса топовой российской дизайнерской студии

---

## 1. Логика последовательности работы в проекте

**Правило студии:** Сначала строится весь объём — мебель, полы, потолки, развёртки с размерами. Только после этого заполняются остальные планы (коммуникации, розетки, освещение и т.д.).

**Порядок построения:**
1. Напольные покрытия
2. Потолок
3. Отделка стен (план разверток)
4. Расстановка мебели и сантехники
5. Фасады (развертки) с размерами
6. После → коммуникации, розетки, освещение

**Почему так:** Коммуникации нельзя спроектировать без расстановки мебели. Развёртки с точными привязками — основа для всех остальных планов.

---

## 2. Привязка плана на листе

На каждом листе в шаблоне есть скрытая группа-маяк — точка, к которой привязывается план.

**Как использовать:**
1. Включить "Показать скрытые объекты" (лампочка)
2. Сдвинуть план к маяку (угол плана = угол маяка)
3. Выключить лампочку

**Зачем:** Планы не "бегают" по листам — каждый лист начинается с одной точки, проект читается чище.

## 3. Напольные покрытия

### Типы перекрытий в шаблоне

| Тип | Пирог (толщина) | Применение |
|---|---|---|
| Инженерная доска | 130 мм | Жилые комнаты |
| Плитка под дерево (тонкая) | 130 мм | Прихожая, кухня |
| Плитка под дерево (толстая) | 250 мм | Там, где нужно выравнивание |
| Белая плитка (тонкая) | 130 мм | Санузел — основной |
| Белая плитка (толстая) | 250 мм | Санузел — с выравниванием |

**Важно:** Если плитка 1200×1200 и в душе режется на полосы 200 мм — штриховка меняется, маркировка остаётся одна → в спецификации считается всё вместе.

### Построение перекрытия
1. Создать аналог от нужного типа
2. Обрисовать контур по стенам
3. Для быстрого замыкания углов — "Обрезать/Удлинить до угла"
4. Проверить отметку: для чистого пола → 0
5. Поставить привязку к стенам через "Выбор линии" (если стена двинется — перекрытие двинется вместе)
6. Убрать галочку "Граница помещения" у перекрытия

**Конфликт толщин:** Если Revit ругается на пересечение — уменьшить толщину перекрытия на 10 мм (убрать 10 мм из пирога).

### Направление укладки
Выделить штриховку через Tab → "Повернуть" → 90° (привязать к направлению на окно или вдоль помещения).

## 4. Потолок

### Типы потолков в шаблоне

| Тип | Применение |
|---|---|
| Потолок белый | Основной — жилые комнаты |
| Потолок тёмный (черный/голубой) | Акцентный — санузел, спальня |
| Натяжной / световой | Переименовать в спецификации |

### Построение потолка
1. Создать аналог → "Автопостроение" по помещению (если предлагает) или вручную по контуру стен
2. Для сложных форм — использовать "Разделитель": нарисовать одну линию, поставить точку разделения над проёмами и пересечениями
3. Пройти уголком по контуру помещений
4. Выставить отметки по разрезу:
   - Проверить по балкам: потолок ниже нижней грани балки
   - Стандарт: 2700–2760 мм для основного потолка
   - Выступы/ниши: по проекту, min 2600 мм

## 5. Отделка стен

### Принцип
Отделка строится стенами толщиной **1 мм** (покраска) или витражами **20 мм** (МДФ, плитка, текстиль). Всё сводится в одну спецификацию.

### Типы отделки в шаблоне (маркировка OS-1 … OS-13)

| Маркировка | Тип | Толщина | Применение |
|---|---|---|---|
| OS-1 | Покраска белая | 1 мм | Основная, жилые комнаты |
| OS-2 | Покраска чёрная | 1 мм | Акценты |
| OS-3 | Покраска (цвет по проекту) | 1 мм | Любой цвет |
| OS-4 | Плитка | Витраж 20 мм | Санузел, кухня-фартук |
| OS-10 | Текстильные панели | Витраж 20 мм | Спальня, гостиная |
| OS-11 | МДФ панели | Витраж 20 мм | Санузел (закрытие шкафа/инсталляции) |

**Итого:** 13 типов в шаблоне. Покраска — обычные стены 1 мм. Плитка/МДФ/текстиль — витражи.

### Построение отделки
1. Взять OS-1, создать аналог
2. Рисовать аналогично штукатурке — сначала с небольшим отступом от стены
3. После построения — выровнять к стенам через "Выравнивание"
4. "Запретить соединение" — отделка не сливается со стенами
5. Убрать "Границу помещения" у стен отделки (чтобы помещение считалось по монтажным стенам)
6. Проверить привязку по высоте: верх — минус 100 от верхнего перекрытия

### Вырезание дверного проёма в отделке
- Стандартный проём: "Соединить" → выбрать стену отделки → выбрать дверь
- Скрытая коробка: НЕ соединять → на развёртке вырезать проём вручную через "Редактировать профиль"

### Маркировка отделки
- Поставить марку: Аннотации → Марка категории → навести на стену
- Маркировать по одной (не две марки на одну стену)
- После постановки марки — сбросить Esc и взять марку снова

## 6. Расстановка мебели

### Принцип отступа 1 мм
Шкафы у стен ставятся впритык — отступ 0. Шкафы с фасадами на столешницах и столах — отступ 1 мм от грани стены/отделки. Зазор читается на 3D как фасад.

### Создание мебели в контексте (раковина, нестандартные элементы)
1. Архитектура → Компонент → стрелка → "Создать модель в контексте"
2. Категория → Мебель (или Сантехника)
3. Форма: Выдавливание (выбрать внешний контур)
4. Применить материал
5. Для раковины: добавить "Полую форму" → обрисовать контур раковины → смещение 10 мм → завышение 10 мм

### Стиральная машина / скрытые элементы
- Разместить компонент
- ПКМ → "Переопределить графику для элемента" → "Прозрачность" → Применить
- Теперь элемент виден сквозь шкаф

### Верхний душ (невидимый на стандартном плане)
1. Вид → Фрагмент плана → нарисовать область
2. В свойствах фрагмента → Секущий диапазон → изменить высоту сечения на 2400–2500 мм
3. Душевая лейка становится видна

## 7. Оформление плана мебели

### Оси построения
- Продлить основные оси за габариты плана
- Подписать текстом: "Основная линия построения"
- Текст копировать созданием аналога, выравнивать по линии

### Правила простановки размеров

| Тип | Цвет | Что показывать |
|---|---|---|
| Основные габариты | Черный | Общий размер + цепочка по выступам |
| Привязки сантехники | Красный | Оси смесителей, центры унитазов |
| Проходы/коридоры | Черный | Важные проходы |
| Округлённые | Специальный тип "Округлённый" | Там где 1 мм от раковины/стекла |

**Правила:**
- Размеры по одной линии, не "мельчить"
- Выносить за габариты плана
- Размер со звёздочкой (*) = не привязываться при проектировании кухни, указывается справочно
- Проходы — не округлять, писать точные миллиметры
- Размеры сантехники дублировать на план коммуникаций и развертки (красные)

### Выноски элементов
- Тип: "Текст с выноской" (с точкой)
- Для стрелки: нажать кнопку со стрелкой вверху
- За среднюю точку — делать круглообразную выноску

### Что подписывать обязательно
- Все сложные элементы: электрощиток, скрытые в шкафу устройства
- Элементы из ТЗ заказчика: кофемашина, пылесос Dyson, робот-пылесос
- Посудомоечная машина: обозначить серой пунктирной линией (габарит)
- Холодильник, духовой шкаф, кофемашина — подписи

### Что скрыть на плане мебели
- Напольные покрытия → шаблон вида → Фильтры → убрать галочку "Напольные покрытия"
- Это убирает мешающие швы и грани от перекрытий

## 8. Душевые перегородки (стеклянные)

- Делаются витражами (аналогично окнам)
- Толщина стекла: 8 мм
- Для открывающейся двери: открепить системную панель → найти "Дверь без рамки" или "Дверь с рамкой" в спецификации витража
- Размеры двери задаются для каждого экземпляра (не в типе)
- Для вида "без рамки": все параметры рамки = 1

**Зазор для шторки:** 100 мм от края двери.

## 9. Развёртки (Фасады)

### Создание фасадов
1. На плане разверток временно включить мебель: шаблон вида → категории → Мебель + Сантехника → полутона
2. Вид → Фасад → поставить кружок у стены (Revit ориентирует направление)
3. Один фасад = одна стена (один лист = одно помещение, несколько фасадов)
4. Отрегулировать границу фасада по ближайшей стене

### Оформление развёртки
- Масштаб: 1:30 (лист А3)
- Шаблон вида: "Фасады отделка"
- Размеры: все привязки из плана мебели дублируются сюда
- Красные размеры (осевые): оси смесителей, унитаза, инсталляции
- Схема разрезки витража: для коллекторного шкафа, душевой перегородки

### Стандарт отметок на развертке
- Инсталляция: клавиша смыва — 125 мм от каждого края импоста
- Ось гигиенического душа: 300 мм от стены, высота 650 мм
- Ось смесителя раковины: по центру раковины, высота 1000 мм от пола

### Нумерация развёрток
Развёртки нумеруются по номеру помещения: лист "Развертки 4" → санузел (помещение №4).
