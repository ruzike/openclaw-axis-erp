# Naming Standard — Стандарт именования (Revit)

> Источник: Курс топовой российской дизайнерской студии (практика)

---

## 1. Виды (Views)

Виды называются **точно по названию листа**, к которому они относятся.

| Вид | Название в диспетчере |
|---|---|
| Обмерный план | `Обмерный план` |
| Демонтаж | `Демонтаж` |
| Монтаж (этап 1) | `Монтаж 0.1` |
| Монтаж (этап 2) | `Монтаж 0.2` |
| Мебель | `Мебель` |
| Коммуникации | `Коммуникации` |
| Напольные покрытия | `Напольные покрытия` |
| Потолок | `Потолок` |
| Освещение | `Освещение` |
| Развёртки | `Развёртка — [название помещения]` |

**Правило:** Новые виды, созданные вне шаблона, попадают в папку "?" (вопросики). Это сигнал — вид без шаблона вида, нужно назначить шаблон вручную.

---

## 2. Стены

Именование типов стен: `[Материал] [Толщина]`

Примеры:
- `Monolith 300`
- `Monolith 200`
- `Кирпич 250`
- `Блок 300`
- `Теплоизоляция балкон`
- `Звукоизоляция`
- `ГКЛ 75` (50 профиль + 2 слоя ГКЛ)
- `НГКЛ 100` (влагостойкий, санузел)

**Правило:** При создании нового типа стены — сразу переименовать по этому формату.

---

## 3. Группы моделей (Model Groups)

Группы используются для разделения элементов по стадиям и назначению:

| Группа | Назначение |
|---|---|
| `Сущ` (существующая) | Существующие стены, которые остаются |
| `Демонтаж` | Стены под снос — красятся на плане демонтажа |
| `Радиатор` | Радиаторы отопления |
| `Монтаж` | Группа коллекторного узла/шахты — дублируется на всех планах |

---

## 4. Материалы

Кастомные материалы студии маркируются символом `*` в начале названия — для быстрого поиска в начале списка.

Примеры: `*Бетон монолит`, `*Теплоизоляция`, `*Штукатурка`

---

## 5. Линии

Кастомные линии маркируются в начале с пробелом, затем: `цвет — тип — толщина`

Примеры: ` Чёрный пунктир 1`, ` Серый сплошной 3`, ` Фиолетовый сплошной 1`

Системные (программные) линии — в скобках.

---

## 6. Штриховки

Кастомные штриховки — символ `*` или `**` в начале:
- `*` — кастомные именные
- `**` — простые чёрные и серые заливки

---

## 7. Листы

Листы называются **по типу плана + номер**.

Примеры: `А1 — Обмерный план`, `А2 — Демонтаж`, `А3 — Монтаж`

При экспорте PDF — Revit автоматически формирует имя файла: `[Название проекта] — [Номер листа] — [Название листа]`. Допускается переименовывать просто по номеру (`А1`, `А2`) для порядка в папке.

---

## 8. 3D-виды

Каждому плану соответствует 3D-вид с тем же именем:
- `3D Обмер`
- `3D Демонтаж`
- `3D Монтаж 1`
- `3D Монтаж 2`

Размещаются на том же листе, что и план.
