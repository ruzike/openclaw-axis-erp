# BIM QA Checklist — Контроль качества

> Источник: Практика студии + базовые стандарты BIM для интерьерного проекта

---

## 1. Checklist перед сдачей каждого плана

### Общее
- [ ] Вид использует правильный шаблон вида (не в папке "?")
- [ ] Масштаб установлен корректно (1:50 или 1:40 для плана)
- [ ] Обрезка вида включена, граница убрана, аннотации обрезаны
- [ ] Легенда на листе соответствует содержимому плана (лишние элементы убраны)
- [ ] Все элементы промаркированы (помещения, двери)

### Стены
- [ ] У всех существующих стен: стадия возведения = `Существующая`
- [ ] У демонтируемых стен: стадия возведения = `Существующая`, стадия сноса = `Монтаж`
- [ ] У монтажных стен: стадия возведения = `Монтаж`
- [ ] Привязки по высоте: снизу `Черный пол`, сверху `Второй этаж` (или `Неприсоединённая`)
- [ ] Нет "сливающихся" углов там, где не нужно (проверить соединения)

### Обмерный план
- [ ] Все стены присутствуют по обмерам
- [ ] Указаны высоты всех проёмов (окна, двери, витражи)
- [ ] Указаны высоты выводов вентиляции и воды
- [ ] Указаны площади: помещения, балкон, общая
- [ ] Размерные цепочки: линия + цепочка по выступам + общий
- [ ] Шахта/коллекторный узел обозначен условно

### Монтажный план
- [ ] Двери скрыты (виден только проём)
- [ ] Коллекторный узел: ось инсталляции 450 мм, отступ от стены 700 мм
- [ ] Указаны только ключевые размеры (санузлы, проходы)
- [ ] Помещения расставлены с правильными разделителями
- [ ] Стены 2-го этапа (ГКЛ) не сливаются со стенами 1-го этапа

### Экспорт PDF
- [ ] Выбран режим "Выбранные виды и листы"
- [ ] Масштаб "Сто-процентный" (не "Вписать")
- [ ] Формат соответствует листу (А3/А4)
- [ ] Файл назван по номеру листа

## 2. Типичные ошибки

| Ошибка | Причина | Решение |
|---|---|---|
| Стена не появляется на плане | Неправильная стадия возведения | Проверить и изменить стадию |
| Элемент виден на всех планах | Использована "Линия модели" вместо "Линии детализации" | Перерисовать правильным инструментом |
| Демонтируемые стены не красятся | Фильтр стадий не назначен в шаблоне вида | Настроить фильтры в шаблоне |
| Размер "не доходит" до стены (витраж) | Витраж привязывается к элементам построения, не к грани стены | При простановке размера — Tab для выбора грани стены |
| Вид не печатается / печатается пустым | Выбран режим "Текущее окно" | Переключить на "Выбранные виды и листы" |
| Перекрытие не замыкается по углу | Линии не доведены до пересечения | Использовать "Обрезать/Удлинить до угла" |
| Радиатор не виден на плане | Привязка от чистого пола, секущий диапазон вида с нуля | Поднять радиатор на 1–20 мм |
| Помещение заливает всё пространство | Нет разделителей помещений | Добавить "Разделитель помещений" (включить видимость в шаблоне) |
| Стены "сливаются" на 3D | Не отменено соединение на углах | ПКМ на кружочке угла → "Отменить соединение" |

## 3. Проверка модели перед финальной сдачей

- [ ] Диспетчер связей: нет лишних загруженных файлов (DWG, RVT, PDF)
- [ ] Нет "тяжёлых" семейств с излишней детализацией для данного LOD
- [ ] Все виды имеют шаблоны (нет видов в папке "?")
- [ ] Спецификации актуальны (нет удалённых помещений без строк)
- [ ] Файл сохранён после последних изменений (Ctrl+S)
