# Workflow — Рабочие процессы студии

> Источник: Курс топовой российской дизайнерской студии (практика)

---

## 1. Создание нового проекта

1. Открыть Revit → Создать → Обзор → выбрать шаблон студии `.rte`
2. Выбрать "Проект на основе шаблона"
3. После открытия — настроить рабочее пространство: Диспетчер задач → вправо, открыть все свитки

**Что уже есть в шаблоне:**
- Все виды с именами по листам
- Все листы с вынесенными видами и легендами
- Шаблоны видов для каждого плана
- Кастомные стили линий, штриховок, материалов (с `*`)
- Группа элементов коллекторного узла/шахты (дублируется на всех планах)
- Панель быстрого доступа: сохранение, размеры, марка, линия детализации, штриховка

---

## 2. Обмерный план — построение

### Шаг 1: Настроить отметки
- Открыть разрез
- Выставить: Черный пол = 0.000, Чистый пол = +0.120, Второй этаж = по замеру

### Шаг 2: Построить существующие стены
- Брать стены из шаблона (Monolith 300, Кирпич 250, Блок 300 и т.д.)
- Использовать "Создать аналог" от нужного типа стены
- Стадия возведения → `Существующая`
- Привязка снизу → Черный пол, сверху → Второй этаж

### Шаг 3: Витражи (балконное остекление)
- Основной витраж для балкона: `Стандартный витраж`
- Простое окно: заменить системную панель через "Открепить" → выбрать нужный тип
- Дверь витражная: аналогично через открепление системной панели
- Раздвижная дверь → изменить импосты под размер двери

### Шаг 4: Демонтируемые стены
- Брать стены из группы `Демонтаж`
- Стадия возведения → `Существующая`, стадия сноса → `Монтаж`
- "Запретить соединение" — чтобы не сливались с остающимися стенами

### Шаг 5: Перекрытие существующее
- Создать аналог от существующего перекрытия
- Привязка → Черный пол
- Стадия → `Существующая`
- Использовать "Обрезать/Удлинить до угла" для замыкания контура

### Шаг 6: Радиаторы
- Брать из группы `Радиатор на стены` — есть разные виды
- Стадия → `Существующая` (если сносятся → стадия сноса `Монтаж`)

---

## 3. Демонтажный план

1. Открыть лист `Демонтаж` — вид уже вынесен
2. Установить стадию вида → `Монтаж` → фильтры автоматически покрасят демонтируемые элементы
3. Добавить размеры по демонтируемым стенам (для строителей — просчёт вывоза мусора)
4. Добавить 3D вид демонтажа → скопировать от обмерного, переименовать `Демонтаж`
5. Снести балкон (если нужно): выделить стены через фильтр → стадия сноса `Монтаж`

---

## 4. Монтажный план — Этап 1 (выравнивание)

**Что входит в Этап 1:**
- Выравнивающая штукатурка по существующим стенам
- Новые стены из блоков/кирпича
- Теплоизоляция (балкон)
- Звукоизоляция (периметр, соседи)

**Построение штукатурки:**
1. Взять стену выравнивания
2. Поставить на небольшом отступе от существующей стены
3. После рисования всего периметра — выровнять к существующим стенам через "Выравнивание"
4. Привязка: внутренняя чистовая поверхность (чтобы толщина добавлялась вовнутрь)
5. "Запретить соединение" — штукатурка не должна сливаться с существующими стенами

**Важно для монтажных размеров:**
- Ставить только ключевые размеры: габариты санузлов, ширина проходов, коридоров
- НЕ ставить дверь на плане монтажа — только проём (строители иначе путают проём и размер двери)
- Скрыть фильтр "Двери для монтажа" в шаблоне вида

---

## 5. Монтажный план — Этап 2 (ГКЛ, санузел)

**Что входит в Этап 2:**
- Стены из ГКЛ (`ГКЛ 75`)
- Влагостойкие стены санузла (`НГКЛ 100`)
- Инсталляция (унитаз)
- Коллекторный узел

**Стандарт коллекторного узла:**
- Отступ от стены → 700 мм
- Ось инсталляции → 450 мм от края
- Высота инсталляции → 1200 мм от уровня чистого пола
- Нижняя закрывающая стенка → 25 мм (чтобы было видно инсталляцию)
- Разделение в зоне узла: низ — ГКЛ глухой, верх — шкаф МДФ

## 6. Оформление листа

### Масштаб
- Обмерный план: обычно 1:50 или 1:40
- Детали/узлы: 1:20 или 1:10

### Аннотации на обмерном плане
- Размерные цепочки: одна линия выносок + цепочка по выступам + общий размер
- Высоты проёмов: выноска с "нижним колонтитулом" — высота подоконника / высота проёма
- Площади: площадь помещения, балкона, общая

### Условные обозначения
- Легенды вынесены на каждый лист по типу плана (монтаж, демонтаж, мебель, полы, освещение)
- Убрать из легенды всё, что не используется на данном плане

### Обрезка вида
1. Включить "Обрезать вид" + "Показать границу"
2. Настроить границу
3. Включить "Обрезать аннотации"
4. Убрать "Показать границу"

## 7. Экспорт PDF

1. Файл → Печать
2. Принтер: Adobe PDF (или Microsoft Print to PDF)
3. Выбрать: "Выбранные виды и листы" → кнопка "Выбрать" → галочка на "Листы" → выбрать нужный
4. Формат: А3 (или нужный)
5. Масштаб: "Сто-процентный масштаб" (не "Вписать") — чтобы не уменьшался
6. Ориентация: книжная / альбомная по листу
7. Имя файла: по номеру листа (`А1`, `А2`) для порядка в папке

## 8. Полезные приёмы (Tips & Tricks)

| Действие | Способ |
|---|---|
| Мозаичное расположение окон | `WT` на клавиатуре |
| Создать аналог элемента | Выделить → "Создать аналог" |
| Копировать свойства (материал, тип) | Выделить источник → "Копировать свойства" → выделить цель |
| Выделить все элементы одного типа на виде | ПКМ → "Выделить все на виде" |
| Убрать элемент из выделения | Shift + клик |
| Выделить системную панель витража | Tab |
| Закрепить элемент (нельзя двигать) | Кнопка "Прикрепить" → удобно при работе с освещением |
| Вставить скопированное на тот же вид | Вставить → "По текущему виду" |
| Соединить геометрию двух стен | Кнопка "Соединить" → первая стена → вторая |
| Разъединить угол стен | ПКМ на кружочке угла → "Отменить соединение" |
| Убрать элемент из проекта полностью (связь) | Диспетчер связей → Удалить |
