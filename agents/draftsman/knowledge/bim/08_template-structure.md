## 9. Важные правила для агента

1. **Листы уже созданы** — не создавать новые листы с теми же именами
2. **Шаблоны видов уже настроены** — назначать из списка выше, не создавать новые
3. **Семейства уже загружены** — использовать из списка, не загружать дубли
4. **Материалы** — использовать только `*` материалы студии
5. **Стены** — типы стен берутся из шаблона, новые создаются по стандарту именования `01_naming-standard.md`
6. **Уровни** — три уровня зафиксированы, не создавать новые без необходимости
