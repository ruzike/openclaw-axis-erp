# Стадии проекта (Stages) и LOD — Детализация элементов

> Источник: Курс топовой российской дизайнерской студии (практика интерьерного дизайна)

---

## 1. Стадии Revit (Phasing)

В интерьерном проекте используются **2 стадии**:

| Стадия | Название в Revit | Что включает |
|---|---|---|
| Стадия 1 | `Существующая` | Всё что есть до начала работ |
| Стадия 2 | `Монтаж` | Всё новое — демонтаж, перегородки, отделка |

### Логика стадий для каждого элемента:

| Тип элемента | Стадия возведения | Стадия сноса |
|---|---|---|
| Существующая стена (остаётся) | Существующая | — |
| Демонтируемая стена | Существующая | Монтаж |
| Новая стена (монтаж) | Монтаж | — |
| Существующий радиатор (сносится) | Существующая | Монтаж |
| Перекрытие существующее | Существующая | — |

**Важно:** Демонтируемые стены — отдельная группа моделей `Демонтаж`. Это позволяет быстро выделять их через фильтр, не задевая другие стены.

## 2. Планы проекта — последовательность

Полный комплект планов интерьерного проекта (от обмеров до финала):

```
01. Обмерный план          — существующее состояние
02. План демонтажа         — что сносим
03. План монтажа (этап 1)  — выравнивание, блоки, теплоизоляция
04. План монтажа (этап 2)  — ГКЛ, санузел, НГКЛ
05. Мебель                 — расстановка, кухня, санузел
06. План заполнения проёмов — двери, окна
07. Коммуникации           — подводы воды, канализация
08. Радиаторы              — отопление
09. Напольные покрытия
10. Плинтусы
11. Тёплый пол
12. Потолок
13. Узлы по потолку
14. Теневые зазоры
15. Освещение
16. Маркировка освещения
17. Вентиляция
18. Управление светом
19. Розетки
20. Отделка стен
21. Развёртки
22. Оформление развёрток + молдинги
```

## 3. Детализация (LOD) по стадиям

### Обмерный план (LOD 200)
- Стены: геометрия по реальным обмерам
- Окна/двери: витраж + тип открывания, указание высоты проёма
- Радиаторы: позиция, тип (настенный/встроенный)
- Аннотации: габариты помещений, высота подоконников, высоты проёмов
- Площади: помещения, балкон, общая
- Инженерия: условное обозначение выводов воды/вентиляции с высотами

### Монтажный план (LOD 300)
- Стены: с разбивкой на этапы (этап 1 — штукатурка/блоки/утепление, этап 2 — ГКЛ)
- Проёмы: указывается только размер проёма, без двери — чтобы строители не путали
- Размеры: только ключевые — габариты санузлов, проходы, коридоры
- Коллекторный узел: позиция, ось инсталляции (450 мм от края)
- Радиаторы: тип, размер, привязка по высоте

### Планы отделки (LOD 300)
- Помещения: выставлены с площадями и названиями
- Материалы: штриховка по типу покрытия
- Маркировка: каждый элемент промаркирован

## 4. Уровни (Levels) — стандарт отметок

| Уровень | Отметка | Примечание |
|---|---|---|
| Черный пол | 0.000 | Нижняя плита перекрытия |
| Чистый пол | +0.120 | Стяжка ~120 мм (по отметке МОП/холла) |
| Второй этаж | По замеру | Верхнее перекрытие |

**Правило:** Все существующие стены привязываются: снизу — Черный пол, сверху — Второй этаж (уровень перекрытия). Новые стены монтажа — аналогично.

Радиаторы встроенные: привязка от черного пола + 20 мм (чтобы был виден в сечении вида).

## 5. Шаблоны видов (View Templates)

Каждый вид использует **свой шаблон** с настроенной видимостью категорий.

Что контролируется в шаблоне:
- Видимые/скрытые категории элементов (стены, двери, мебель, коммуникации)
- Фильтры по стадиям — для демонтажа включается фильтр `Стадия: Монтаж`, элементы красятся по стадии
- Фильтры категорий — например, скрыть `Двери для монтажа` (чтобы строители видели только проём)
- Аннотации: разрезы включаются/выключаются по надобности

**Правило:** Если вид создан вне шаблона — он попадает в папку "?". Нужно назначить шаблон вручную из существующих.
