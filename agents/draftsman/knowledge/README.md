# Draftsman Knowledge Base — Навигация

Всё что нужно для создания безупречных чертежей: стандарты, правила, чек-листы.

---

## 📋 Оглавление

### 1. **layer-standards.md**
Таблица слоев (Layer Standards): имена, цвета, веса линий.
- Архитектурные слои (стены, двери, окна)
- Интерьер и мебель
- Аннотации (размеры, тексты, оси, штампы)
- Жесткие правила (`ByLayer` всегда!)

### 2. **dimension-rules.md**
Правила простановки размеров (Dimensions):
- Стили размеров (DIMSTYLE)
- Цепочки (внешние и внутренние)
- Правила выравнивания
- Запреты (Override текста размеров)

### 3. **check-list.md**
Чек-лист самопроверки перед сдачей:
- Проверка геометрии (Purge, Audit, единицы)
- Проверка слоев (ByLayer, правильные слои)
- Проверка размеров (стиль, цепочки, габариты)
- Проверка листов (масштаб, штамп, рамка)
- Проверка PDF (качество, веса линий, читаемость)
- Критические ошибки (автоматический отказ)

---

## 🎯 Быстрый старт

**Получил задачу от Ops?**
1. Открыть `layer-standards.md` → Создать/проверить слои
2. Построить геометрию → Все примитивы строго на правильных слоях
3. Проставить размеры → `dimension-rules.md` → Цепочки, стиль `AXIS_DIM_100`
4. Оформить листы → Штамп, рамка, масштаб в Viewport
5. Самопроверка → `check-list.md` → Пройти все пункты
6. Экспорт PDF → Проверить качество
7. Отчет → Отправить в Ops

**Сдача работы:**
→ `check-list.md` → Формат отчета → Отправить DWG + PDF

---

## ⚠️ ЖЕСТКИЕ ПРАВИЛА (НИКОГДА НЕ НАРУШАТЬ)

1. **ByLayer всегда:** Цвет, тип линии, вес — строго `ByLayer`.
2. **Никакого Override:** Размерный текст = реальная геометрия.
3. **Purge + Audit перед сдачей:** Обязательно.
4. **Единицы — миллиметры:** Никаких метров на строительных планах.
5. **Слои по стандарту:** Таблица из `layer-standards.md` — закон.
