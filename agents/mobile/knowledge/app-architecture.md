# AXIS Mobile App — Архитектура

## Экраны

### 1. Главный (Dashboard)
- KPI балл (общий %)
- Просроченные задачи (число)
- Кто заполнил брифинг
- Здоровье агентов
- Быстрые действия

### 2. Чат с агентами
- Список 12 агентов (Main, Ops, Tech, DevOps, Sales, Finance, Shket, HR, QC, Draftsman, Team, Strategy)
- Выбрал агента → чат (WebSocket)
- История сообщений
- Голосовые сообщения (STT → текст → агент → TTS)

### 3. Trello Board
- Карточки по спискам (Бэклог, Очередь, В работе, Готово)
- Цветовая индикация (просрочено = красный)
- Клик → детали карточки
- Drag-and-drop для перемещения

### 4. Команда
- Руслан, Мирас, Бахытжан
- Статус брифинга (заполнил/нет)
- KPI каждого
- Ритуалы

### 5. AI Офис
- Визуализация офиса (как office.html но нативно)
- Клик на агента → детали

### 6. Настройки
- Push-уведомления (вкл/выкл по агенту)
- Тема (тёмная/светлая)
- Авторизация

## Навигация
Bottom Tab Bar: Dashboard | Чат | Trello | Команда | Офис

## API Endpoints (OpenClaw Gateway)
- `GET /api/status` — статус системы
- `POST /api/sessions/send` — сообщение агенту
- `GET /api/cron/list` — список кронов
- `WS /ws` — WebSocket для real-time чата
