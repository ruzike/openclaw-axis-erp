## 📋 TRELLO РАБОТА С КАРТОЧКАМИ

### 1. ДОБАВЛЕНИЕ НОВОЙ КАРТОЧКИ

**Правило:** Только если задача не может быть добавлена к существующей.

**Процесс:**

1. **Проверить доску C3 — Рабочий проект**
   ```bash
   trello.py list C3-Rabochiy-proekt
   ```

2. **Выбрать подходящую колонку:**
   - `📋 Задачи` — текущие задачи
   - `🏗️ В работе` — задачи в разработке
   - `✅ Готово` — завершённые задачи
   - `📊 Отчёты` — отчётные документы

3. **Создать карточку:**
   ```bash
   trello.py add-card C3-Rabochiy-proekt "📋 Задачи" \
     "Название задачи" \
     "Описание: ... ..." \
     "due": "2026-02-25"
   ```

4. **Указать владельца:**
   - Тег: @Мирас (дизайн), @Бахытжан (3D)
   - Или тег: @разработчик (общий)

5. **Добавить ссылки:**
   - Google Doc, Figma, Sketch
   - Чат команды, Telegram группа

### 2. ОБНОВЛЕНИЕ СТАТУСА

**Правило:** Обновлять по факту, не по расписанию.

**Процесс:**

1. **Проверить актуальность карточки**
   ```bash
   trello.py read <card_id>
   ```

2. **Переместить карточку:**
   - `📋 Задачи` → `🏗️ В работе` (начал работу)
   - `🏗️ В работе` → `✅ Готово` (завершил)
   - `🏗️ В работе` → `📋 Задачи` (отложил)

3. **Отметить владельца:**
   - Команда отмечена в комментарии: "Начал работу"

### 3. ЗАПИСЬ ОТЧЁТА

**Правило:** Отчёт по задаче — это отдельная карточка в разделе `📊 Отчёты`.

**Процесс:**

1. **Создать карточку:**
   ```bash
   trello.py add-card C3-Rabochiy-proekt "📊 Отчёты" \
     "Отчёт по задаче: ..." \
     "Статус: ✅ Готово" \
     "Результат: ..." \
     "Баги: ... (если есть)"
   ```

2. **Добавить чеклист:**
   ```bash
   trello.py add-checklist <card_id> "Что проверено" \
     "✅ ТЗ согласовано" \
     "✅ Время согласовано" \
     "✅ Ресурсы выделены" \
     "✅ Контрольный срок соблюдён"
   ```
