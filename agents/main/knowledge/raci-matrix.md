# Матрица RACI (AXIS)
**Версия:** 2.0 | **Дата:** 2026-03-09

Матрица распределения ответственности в гибридной команде (Люди + AI Агенты).
Используй эту таблицу, чтобы точно знать, с кого спрашивать результат (A), кому ставить задачу (R), с кем советоваться (C) и кого просто уведомлять (I).

## 🔑 Расшифровка ролей
- **R (Responsible / Исполнитель):** Тот, кто делает задачу руками.
- **A (Accountable / Владелец):** Тот, кто отвечает "головой" за результат. Принимает работу. **Всегда только ОДИН человек/агент на процесс.**
- **C (Consulted / Консультант):** Эксперт. С ним советуются *до* принятия решения или в процессе.
- **I (Informed / Информируемый):** Тот, кого ставят перед фактом после выполнения (получает дайджест или статус).

## ⚠️ Правила применения RACI для Агентов
1. **Запрет на пинг-понг:** Агент с ролью "A" (Accountable) не имеет права сбрасывать ответственность на агента "R". Если "R" не справляется, "A" эскалирует проблему Руслану.
2. **Делегирование:** Задачу всегда ставит "A" для "R".
3. **Согласования:** Если агент находится в роли "C", он обязан предоставить данные по запросу (например, Finance выдает рентабельность для Sales).
4. **Exmain** — архивный агент, в матрице не участвует.
