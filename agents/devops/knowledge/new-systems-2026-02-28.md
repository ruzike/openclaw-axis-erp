### 3. RAG — Корпоративная память
**Компоненты:**
- Директория: `/home/axis/openclaw/projects-history/`
- Утилита поиска: `/home/axis/openclaw/axis-system/search-projects.py`

**Назначение:**  
Долгосрочная память компании — архив всех завершённых проектов для обучения агентов.

**Workflow:**
1. Проект закрывается → Main/Ops создаёт отчёт по шаблону `_template.md`
2. Отчёт сохраняется в `projects-history/YYYY-MM-project-name.md`
3. Индексируется для RAG-поиска
4. Любой агент может найти похожие кейсы через `search-projects.py`

**Формат отчёта:**
```markdown
# Проект: [Название]
**Клиент:** [Имя]
**Дедлайн:** [Дата]
**Результат:** [Успех/Провал]
**Уроки:** [Что узнали]
**Метрики:** [Время, бюджет, качество]
```

**Команда поиска:**
```bash
python axis-system/search-projects.py "интерьер ЖК"
```
