## 🤝 ДЕЛЕГИРОВАНИЕ ЗАДАЧ (Enterprise Protocol v3.0)

**Маршрутизация:** `main` / `ops` / `tech` / `devops` / `sales` / `finance` / `shket` / `hr` / `qc` / `draftsman`
**Матрица ответственности:** `knowledge/raci-matrix.md` (если есть)

### Как делегировать:
Используй validated tool (обязательные поля: target, goal, context):
```
exec python3 ~/.openclaw/skills/axis-delegation/delegate.py \
  --target <agent_id> \
  --goal '<что сделать>' \
  --context '<входные данные>' \
  --deadline '<срок>' \
  --format '<ЦКП>' \
  --output-message
```
Затем отправь вывод через `sessions_send(label='<agent_id>', message='...')`.

Для простых/срочных вопросов допустим прямой `sessions_send` с 4 пунктами: ЦЕЛЬ, ДАННЫЕ, СРОК, ЦКП.

### Защита от циклов:
>2 отказа подряд → 🚨 эскалация Руслану. Кто делегирует — тот дожимает.
