## 📄 СЧЁТ (стандартный)

```
═══════════════════════════════════════
         СЧЁТ НА ОПЛАТУ №[номер]
═══════════════════════════════════════

Дата: [дата]
Клиент: [Имя / Компания]
Проект: [название]

───────────────────────────────────────
УСЛУГИ:
───────────────────────────────────────

1. [Описание услуги]
   Кол-во: [N] ракурсов
   Цена: $[цена/ракурс]
   Сумма: $[итого]

2. [Доп. услуги]
   Сумма: $[итого]

───────────────────────────────────────
ИТОГО:                    $[сумма]
Скидка ([X]%):           -$[скидка]
───────────────────────────────────────
К ОПЛАТЕ:                 $[финал]
═══════════════════════════════════════

УСЛОВИЯ ОПЛАТЫ:
• Предоплата 50%: $[сумма] — до [дата]
• Остаток 50%: $[сумма] — при сдаче

РЕКВИЗИТЫ:
[Заполнить реквизиты AXIS]

Счёт действителен 7 дней.
```

---

## ✅ ПРАВИЛА ВЫСТАВЛЕНИЯ СЧЕТОВ

### Сроки:
- Выставить счёт **<24 часа** после согласования КП
- Напоминание об оплате: через **3 дня** после выставления
- Эскалация: через **7 дней** → Руслан

### Обязательные поля:
- [ ] Номер счёта (формат: AXIS-[год]-[номер])
- [ ] Дата
- [ ] Клиент (имя + компания)
- [ ] Проект
- [ ] Детализация услуг
- [ ] Итоговая сумма
- [ ] Условия оплаты
- [ ] Реквизиты

### Нумерация:
- Формат: `AXIS-2026-001`, `AXIS-2026-002`, ...
- Нумерация сквозная за год

## 🔔 ТРЕКИНГ ОПЛАТ

| Статус | Действие |
|--------|----------|
| ✅ Оплачен | Зафиксировать в учёте |
| ⏳ Ожидает | Напоминание через 3 дня |
| ⚠️ Просрочен (7+ дней) | Эскалация → Руслан |
| 🔴 Просрочен (14+ дней) | Приостановить работу |
