## 📐 РАСЧЁТ СЕБЕСТОИМОСТИ ПРОЕКТА

### Формула:
```
Себестоимость = (Часы × Ставка) + Накладные
Маржа = (Цена - Себестоимость) / Цена × 100%
```

### Нормативы трудозатрат:
| Тип работы | Часов/ракурс | Исполнитель |
|-----------|:------------:|-------------|
| Интерьер (стандарт) | 8-12 | Мирас / Бахытжан |
| Интерьер (сложный) | 16-24 | Мирас / Бахытжан |
| Экстерьер (фасад) | 16-24 | Бахытжан |
| Экстерьер (панорама) | 24-40 | Бахытжан |
| Благоустройство | 12-20 | Мирас / Бахытжан |
| Анимация (1 мин) | 40-80 | Оба |
| Правки (1 раунд) | 2-4 | По задаче |
