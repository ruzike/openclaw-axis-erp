# Knowledge Base — Finance Agent (CFO)
**Версия:** 1.0 | **Дата:** 2026-02-28

---

## 📚 Структура файлов

### 1. pricing.md
**Содержание:** Прайс-лист AXIS, тарифы, скидки, пакеты
**Когда использовать:** При расчёте стоимости проекта, проверке маржинальности

### 2. cost-structure.md
**Содержание:** Структура затрат, зарплаты, накладные, маржинальность
**Когда использовать:** При расчёте себестоимости, бюджетировании, P&L

### 3. invoice-templates.md
**Содержание:** Шаблоны счетов, акты, правила выставления
**Когда использовать:** При выставлении счёта клиенту

### 4. financial-reports.md
**Содержание:** Шаблоны отчётов (Cashflow, P&L, прогноз)
**Когда использовать:** При подготовке еженедельных/месячных отчётов

### 5. clients.md
**Содержание:** Клиенты, контракты, условия оплаты
**Когда использовать:** При расчёте маржи по клиенту, трекинге оплат

### 6. agent-communication.md
**Содержание:** Протокол общения с другими агентами
**Когда использовать:** При межагентном взаимодействии

---

## ⚡ Быстрый старт

- **Расчёт стоимости** → `pricing.md` + `cost-structure.md`
- **Выставить счёт** → `invoice-templates.md`
- **Подготовить отчёт** → `financial-reports.md`
- **Написать агенту** → `agent-communication.md`
