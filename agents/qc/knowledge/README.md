# Knowledge Base — QC Agent (ОКК)
**Версия:** 1.0 | **Дата:** 2026-02-28

## 📚 Структура файлов

1. **checklists.md** — Детальные чек-листы для проверки рендеров (камера, свет, материалы) и чертежей. Главный инструмент проверки качества.
2. **cad-standards.md** — Жесткие стандарты для САПР (AutoCAD/Revit).
3. **violations-guide.md** — Типы нарушений, формат журнала, пороги эскалации.
4. **digests-templates.md** — Шаблоны дайджестов и уровни решений.

## ⚡ Быстрый старт
- Пришел рендер на проверку? Открой `checklists.md` -> блок "ВИЗУАЛИЗАЦИЯ".
- Пришел DWG/PDF чертеж? Открой `checklists.md` -> блок "ЧЕРТЕЖИ" + сверь с `cad-standards.md`.
- Нашел ошибку? Зафиксируй по правилам из `violations-guide.md`.
