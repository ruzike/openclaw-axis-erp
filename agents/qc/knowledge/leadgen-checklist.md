# QC — Чек-листы лидогенерации
# Примечание: FUNNEL, SALES, MARKETER работают на отдельном сервере маркетинга.
# Этот чек-лист используется при аудите маркетингового инстанса.

## ЗОНА 1: Лендинг
- [ ] Заголовок содержит точку кипения золотого сегмента
- [ ] Оффер по формуле: кто / ситуация / результат / метод / без X / за срок
- [ ] Минимум 2 кейса с цифрами
- [ ] FAQ закрывает топ-3 возражения
- [ ] Форма работает, заявки приходят в FUNNEL
- [ ] Мобильная версия проверена
- [ ] Время загрузки < 3 сек

## ЗОНА 2: FUNNEL
- [ ] Время ответа на заявку: до 5 минут
- [ ] Лид-магнит отправлен всем (100%)
- [ ] Горячие лиды переданы в SALES в течение 30 минут
- [ ] Конверсия заявка → диалог: 30%+
- [ ] Конверсия диалог → КЭВ: 15%+

## ЗОНА 3: SALES
- [ ] Первый контакт с горячим лидом: в течение 2 часов
- [ ] КЭВ проведён по структуре (4 блока, 30 минут)
- [ ] КП отправлено в течение 24 часов после КЭВ
- [ ] Конверсия КЭВ → КП: 80%+
- [ ] Конверсия КП → договор: 30%+

## ЗОНА 4: MARKETER
- [ ] Анализ ЦА содержит минимум 3 сегмента
- [ ] Золотой сегмент определён
- [ ] Тексты лендинга: все 8 блоков
- [ ] Нет стоп-слов в текстах
