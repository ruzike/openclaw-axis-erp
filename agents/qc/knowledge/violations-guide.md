# Типы нарушений и фиксация

## Формат каждой записи в журнале
- **Что нарушено?** (конкретное правило/регламент)
- **Кто нарушил?** (имя)
- **Когда?** (дата, время)
- **Последствия?**
- **Системная проблема или человеческий фактор?**

## Типичные нарушения

### Проектное управление
- Задача не записана в Trello
- Задача без срока/ответственного
- Задача не по формуле делегирования
- Опоздание на планёрку
- Не заполнена форма план-отчёта
- Протокол не отправлен в течение 24 часов

### Коммуникации
- Не ответил в течение 4 часов (рабочее время)
- Грубость с клиентом/коллегой
- Обещал — не сделал без уведомления

### Качество работы
- Рендер не соответствует ТЗ
- Ошибки (свет, материалы)
- Пропущенные правки клиента
- Несоблюдение брендбука

### Процессы
- Нарушение регламента
- Не обновил статус в Trello
- Пропустил дедлайн без уведомления

## Автоматические пороги
- Категория >3 косяков за неделю → эскалация
- Один человек >5 косяков за неделю → эскалация в HR

## Эскалация
- **Системная проблема** → tech (новый регламент)
- **Человеческий фактор** → hr (обучение/коучинг)
- **Критичное** → Руслан напрямую

## 🔄 PDCA-ПЕТЛЯ (закрытие нарушений)

Каждое нарушение в журнале обязано пройти полный цикл:

**PLAN** → зафиксировал нарушение, определил причину (системная/человеческая)
**DO** → отправил замечание нарушителю / задачу tech или hr
**CHECK** → через 3 дня проверяю: устранено ли нарушение?
  - Команда: `find /home/axis/openclaw/agents/*/MEMORY.md -mtime -3` + проверка sessions_history
  - Для нарушений команды (люди): проверить через Ops есть ли повтор в Trello
**ACT** → 
  - Устранено → закрыть запись в журнале статусом ✅ ЗАКРЫТО
  - НЕ устранено → эскалация на уровень выше (1-е нарушение → замечание, 2-е → HR, 3-е → Руслан)

### Формат записи в журнале (обязательно!)
```
## [ДАТА] Нарушение #N
- **Что:** [описание]
- **Кто:** [агент/сотрудник]
- **Тип:** Системная / Человеческая
- **Действие:** [что отправил, кому]
- **Статус:** 🔴 ОТКРЫТО / ✅ ЗАКРЫТО [дата закрытия]
- **Проверка (CHECK):** [дата проверки + результат]
```

### Еженедельный дайджест (воскресенье)
Включать в дайджест:
- Открытые нарушения (не закрытые >7 дней) → 🔴
- Закрытые за неделю → ✅
- Системные паттерны (одна и та же ошибка повторяется) → 🟡
