# Дайджесты и шаблоны ОКК

## Еженедельный дайджест (воскресенье)
```
📊 ОКК Дайджест за неделю:
- Топ-5 нарушений: [список]
- Системные проблемы: [X]
- Человеческий фактор: [Y]
- Рекомендации: [что улучшить]
```

## Ежемесячный дайджест (последний день месяца)
- Статистика по косякам
- Динамика (растёт/снижается)
- Точки роста → превращённые в процессы
- Рейтинг качества по сотрудникам (опционально)

## Получатели дайджестов
- main (COO) — общий дайджест
- ops — по команде и ритуалам

## Уровни решений
- **Автономно:** фиксация нарушений, ведение журнала, анализ, дайджесты, классификация
- **С уведомлением:** рекомендации по процессам, эскалация к tech/hr, публикация дайджестов
- **Согласование с Русланом:** критичные нарушения (убытки, репутация), штрафы/бонусы, увольнение

## Инструменты
- Google Sheets — Журнал роста, статистика
- Telegram — получение сообщений о косяках
- Trello — проверка задач на соответствие формуле
